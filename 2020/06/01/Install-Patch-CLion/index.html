破解软件-破解C++工具

说到C++的IDE，我们想到的大都是Dev-C++，因为Dev-C++是一个十分轻量级的IDE，而且许多教科书、教学课程上都会默认使用Dev-C++来编写程序。不过，虽然它比较好用，但已经停更，现在也只能由Orwell公司来更新，整个软件的界面也不怎么好看，而且程序员里是没有几个持续用Dev-C++的，有也是大神中的大神。而今天我们用的IDE，是由JetBrains公司开发的一款基于Java的IDE——CLion。CLion 是一款专为开发C及C++所设计的跨平台IDE，它有一个极大的益处就是有强大的自动补全功能，写起代码轻松快捷。本文就来演示Windows10x64下如何安装并配置CLion。

1. 准备工作

​ 1.1. 在官网下载，无需扶那啥玩意。

​ 1.2. 下载MinGW编译器，本文提供64位版：蓝奏云盘传送门不得不吐槽一下百度云，太恶心了，除了SVIP下载速度只有几KB/s。

​ 1.3. 下载CLion的破解补丁：蓝奏云盘传送门

2. 开始工作

2.1. 安装CLion

​ 管理员身份运行1.1步下载的安装文件，中途会让你选择与哪些文件加关联与是否创建桌面快捷方式等等

2.2. 配置MinGW

​ 把1.2步下载的压缩包解压到你喜欢的目录下。

​ 打开控制面板，搜索“编辑”，找到编辑系统环境变量，打开选择环境变量，如图：（我对电脑做过美化，可能看着不一样，但这是实实在在的Windows10）

​

在打开的窗口中选中系统变量中的“Path”，点击“编辑”

在弹出的窗口中点击右上角的“新建”

然后把你1.2步下载下来的压缩包的解压路径填进去，加上”\bin”，双引号不要。完成之后如上图所示。此时，MinGW64已被添加到了环境变量中。

2.3. 破解&配置CLion

打开安装好的CLion，会让你设置界面主题等等。接下来，会跳出来一个名为License Activation的窗口，在上方选择Evaluate for free(免费试用)，点击蓝色的按钮Evaluate继续。

如果跳出来一个类似下图的窗口，关掉不要管他，一会配置。

甚好，我心甚喜！朕要打赏！
